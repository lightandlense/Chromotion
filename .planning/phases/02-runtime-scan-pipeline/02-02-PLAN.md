Implement `scan_slice.py` and its full pytest suite `test_scan_slice.py`.

The script takes `rectified_scan.png` (1920x1080) and the `rest_pose_masks/` directory (also 1920x1080), applies each part's alpha mask to the scan, crops tight to the mask bounding box, and writes one RGBA PNG + one `texture_meta_<part>.json` per part. Edge cases (all-transparent mask, all-white scan region) are handled gracefully.

Purpose: This is the second step in the runtime scan pipeline — it converts the full rectified scan into the per-part RGBA textures the Pixi.js renderer loads as sprites.
Output: `src/preprocess/scan_slice.py` (importable + CLI), `tests/preprocess/test_scan_slice.py` (4 test functions, all green)

```python
# Mask load + resize (NEAREST to preserve binary alpha — no gray fringe)
from PIL import Image
mask_pil = Image.open(mask_path)   # RGBA, 1920x1080 (same as scan — no resize needed)
alpha = np.array(mask_pil)[:, :, 3]
alpha_binary = (alpha > 0).astype(np.uint8) * 255  # clamp any gray fringe

# Scan load — OpenCV reads BGR; convert to RGB before PIL RGBA assembly
scan_bgr = cv2.imread(str(scan_path))
scan_rgb = cv2.cvtColor(scan_bgr, cv2.COLOR_BGR2RGB)

# RGBA assembly
rgba = np.zeros((*scan_rgb.shape[:2], 4), dtype=np.uint8)
rgba[:, :, :3] = scan_rgb
rgba[:, :, 3] = alpha_binary

# Tight bounding box
rows = np.any(alpha_binary > 0, axis=1)
cols = np.any(alpha_binary > 0, axis=0)
if not rows.any():
    # All-transparent guard — output 1x1 white RGBA, no exception
    cropped = np.array([[[255, 255, 255, 0]]], dtype=np.uint8)
    crop_x, crop_y, crop_w, crop_h = 0, 0, 1, 1
else:
    rmin, rmax = np.where(rows)[0][[0, -1]]
    cmin, cmax = np.where(cols)[0][[0, -1]]
    cropped = rgba[rmin:rmax+1, cmin:cmax+1]
    crop_x, crop_y = int(cmin), int(rmin)   # top-left offsets in 1920x1080 space
    crop_w, crop_h = int(cmax - cmin + 1), int(rmax - rmin + 1)

# texture_meta schema (locked)
meta = {"part": part_name, "crop_x": crop_x, "crop_y": crop_y, "crop_w": crop_w, "crop_h": crop_h}
```

**Function signature (exact):**
```python
def slice_scan(
    scan_path: Path,
    masks_dir: Path,
    output_dir: Path,
) -> list[str]:
    """
    Slice a rectified scan into per-part RGBA textures using rest_pose_masks.

    Args:
        scan_path: Path to rectified_scan.png (1920x1080 BGR PNG from scan_rectify.py)
        masks_dir: Directory containing <part>.png masks (1920x1080 RGBA, binary alpha)
        output_dir: Directory to write <part>.png and texture_meta_<part>.json files

    Returns:
        List of part names processed (one per mask found in masks_dir)

    Never raises on empty/transparent masks — outputs fallback 1x1 white RGBA texture.
    """
```

**Implementation steps (in order):**

1. Load scan: `scan_bgr = cv2.imread(str(scan_path))`. If None, raise `FileNotFoundError`.
2. Convert: `scan_rgb = cv2.cvtColor(scan_bgr, cv2.COLOR_BGR2RGB)`.
3. Create `output_dir` with `output_dir.mkdir(parents=True, exist_ok=True)`.
4. Find all mask files: `mask_paths = sorted(masks_dir.glob("*.png"))`.
5. For each mask file (part_name = mask_path.stem):
   a. Load: `mask_pil = Image.open(mask_path)` — RGBA. Extract alpha channel.
   b. Resize alpha if needed: if `mask_pil.size != (scan_rgb.shape[1], scan_rgb.shape[0])`, resize with `mask_pil.resize((scan_w, scan_h), Image.NEAREST)`. Since both are 1920x1080, this is a no-op in production but safe.
   c. `alpha_arr = np.array(mask_pil)[:, :, 3]`
   d. `alpha_binary = (alpha_arr > 0).astype(np.uint8) * 255`
   e. Assemble RGBA: `rgba = np.zeros((*scan_rgb.shape[:2], 4), dtype=np.uint8)`, `rgba[:,:,:3] = scan_rgb`, `rgba[:,:,3] = alpha_binary`
   f. Bounding box check:
      ```python
      rows = np.any(alpha_binary > 0, axis=1)
      cols = np.any(alpha_binary > 0, axis=0)
      if not rows.any():
          # All-transparent — fallback
          cropped = np.array([[[255, 255, 255, 0]]], dtype=np.uint8)
          crop_x, crop_y, crop_w, crop_h = 0, 0, 1, 1
      else:
          rmin, rmax = np.where(rows)[0][[0, -1]]
          cmin, cmax = np.where(cols)[0][[0, -1]]
          cropped = rgba[rmin:rmax+1, cmin:cmax+1]
          crop_x, crop_y = int(cmin), int(rmin)
          crop_w, crop_h = int(cmax - cmin + 1), int(rmax - rmin + 1)
      ```
   g. Save texture: use PIL to save RGBA PNG (PIL handles channels correctly):
      `Image.fromarray(cropped, 'RGBA').save(output_dir / f"{part_name}.png")`
   h. Save meta:
      ```python
      meta = {"part": part_name, "crop_x": crop_x, "crop_y": crop_y, "crop_w": crop_w, "crop_h": crop_h}
      (output_dir / f"texture_meta_{part_name}.json").write_text(json.dumps(meta, indent=2))
      ```
6. Return list of processed part names.

**`main()` function:** argparse with `--scan` (required), `--masks-dir` (required), `--output-dir` (required). Print "Sliced N parts to <output-dir>".

**Imports:** `cv2`, `numpy as np`, `json`, `pathlib.Path`, `sys`, `argparse`, `PIL.Image`. No other dependencies.

**CRITICAL — do NOT:**
- Save RGBA output with `cv2.imwrite` (OpenCV does not support RGBA PNG correctly — use PIL for the final RGBA PNG save)
- Loop pixel-by-pixel for masking (use NumPy broadcast)
- Use bilinear/LANCZOS for mask resize (use `Image.NEAREST`)
