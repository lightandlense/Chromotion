# Current Project
Deployment and operations for Color Animals Interactive. Local file installation — no server, no cloud. Everything runs on one Windows machine connected to a projector (scene.html) and webcam (kiosk.html).

# What good looks like
A setup checklist or launch script that gets both screens running in under 10 minutes. Scripts for positioning browser windows on dual-monitor setup. No internet required at runtime.

# What to avoid
Any runtime dependency on internet connectivity. The installation must work fully offline once set up.
