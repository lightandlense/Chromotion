# Current Project
Documentation for Color Animals Interactive installation operators and creature authors.

# What good looks like
- Operator setup guide: how to run both screens, position windows, calibrate the webcam, launch the installation
- Creature authoring guide: how to create new creature templates, spritesheets, and regions.json files
- localStorage message format reference: the contract between kiosk and scene

# What to avoid
Technical implementation details that belong in /planning/specs. This folder is for operator-facing and contributor-facing guides, not internal architecture docs.
